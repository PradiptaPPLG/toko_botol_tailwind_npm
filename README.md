# Sistem Toko Botol

Aplikasi POS (Point of Sale) untuk toko botol dengan manajemen stok multi-cabang, dibuat menggunakan PHP native.

> **Catatan:** Aplikasi ini masih dalam tahap pengembangan.

## Instalasi Cepat

#### Development

```bash
npm run setup:watch
```

#### Production / Testing

```bash
npm run setup:build
```
