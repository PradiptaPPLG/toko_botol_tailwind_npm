## Requirement
- PHP v7.0
- MySQL
- TailwindCSS v3
## Instalasi
Untuk instalasi pastikan sudah menginstall semua requirement
dan ikuti intruksi dibawah ini:
### Instalasi Cepat
#### Development
Jalankan command berikut:
```bash
npm run setup:watch
```
#### Production/Testing
Jalankan command berikut:
```bash
npm run setup:build
```
